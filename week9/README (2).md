#Task Manager Application

* This is a functional and modern Task Manager web application designed to help users organize their daily activities. It features priority settings, due dates, and motivational quotes fetched from an external API


#Features

* Task Management: Add, delete and mark tasks as completed.
* Priority Levels: Categorize tasks as High, Medium, or Low with visual color coding.
* Deadline Tracking: Each task displays a specific due date to ensure timely completion.
* Motivational Quotes: Integrates the ZenQuotes API to display a new inspirational quote on every load.
* Data Persistence: Uses localStorage to save your tasks so they remain available even after refreshing the page.


#Technologies Used

* HTML5 & CSS3: For the semantic structure and dark-themed, responsive user interface.
* JavaScript (ES6+): For DOM manipulation, event handling, and logic.
* Fetch API: To retrieve real-time data from external quote services.


#File Structure

* -index.html: The core structure of the application.
* -style.css: Custom styling for a professional look.
* -script.js: The engine of the application, handling all interactive features.
